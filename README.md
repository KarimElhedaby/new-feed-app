# News Feed App


This is a prototype for Newsfeed that's main focus is to let users find list of articles and navigate between  .


# Key Features

* kotlin
* mvvm-architecture
* repository-pattern
* viewmodel-pattern
* Hilt (DI) 
* data binding
* rxjava3  , rx operators . 
* retrofit
* livedata
* lifecycle - lifecycleobserver
* clean architucture -  multi module
* unit test
